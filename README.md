# fta.tools

Static GitHub Pages landing page for FIRST Technical Advisors.

## Update links

Edit `script.js` and add or update entries in `toolSections`.

Each tool card supports:
- `name`
- `resourceUrl`
- optional `sourceUrl`
- `maintainer`
- `description`
- `tags`

## Request a link

Use the GitHub issue form: `Request a new link`.
